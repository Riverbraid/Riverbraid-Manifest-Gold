# Public Integrity Challenge: Riverbraid v1.5.0-Genesis

Riverbraid does not claim security through obscurity or policy. It claims integrity through mathematics.

## The Challenge
If you can perform any of the following on the v1.5.0-Genesis stationary floor (Root: `fa7475`):
1. **Forge a Proof:** Produce a valid attestation for a modified repository.
2. **Bypass Fail-Closed:** Force the Shield proxy to return unverified data.
3. **Collision:** Produce a matching Merkle root from a different set of repository SHAs.

## Submission
Submit a reproducible proof-of-concept (PoC) to the maintainers or via a public GitHub issue. 

**Confirmed bypasses will be publicly disclosed and integrated into the stationary audit log.**
